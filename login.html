Instructions



Directories

Type in your desired directory name in the add directory box. Then press Submit. This will add a directory to your current directory


Files

Click on the add file button and select the file you want to upload. This will be confirmed by the success message. You can then press submit and this will add a file to your current directory. If you try to add duplicate files or directories, Nothing will happen, but the duplicate will not be added.


Deleting & Moving

To delete a file, simply press the red delete button. To delete a directory, you must first ensure that this directory is empty. Delete any files or other directories located inside it first. To move to another directory, double click on the name of the directory and you will move to it. To move back up to the parent, click on the ../ and you will return to the parent.
